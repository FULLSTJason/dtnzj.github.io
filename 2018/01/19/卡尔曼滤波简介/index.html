5个核心公式

预测部分：

$$
K = Pp H’ ( H Pp H’ + R)^-1;
Xu = ( I - K H ) Xp + K x;
Pu = ( I - K H ) * Pp;
$$

K  = Pp * H' * ( H * Pp * H' + R)^-1;
Xu = ( I - K * H ) * Xp + K * x;
Pu = ( I - K * H ) * Pp;

拟合部分：

Xp = F * Xu;
Pp = F * Pu * F' + Q;
